🎵 Promptify App

AI-Powered Spotify Integration with MCP Tools

🚀 Quick Start

Step 1: Authenticate with Spotify

Click the button below to authenticate with your Spotify account:

🔐 Login with Spotify

Step 2: Ask Questions About Your Music

Use the API to ask questions about your Spotify data:

POST /ask
{
  "prompt": "What are my top 5 artists?",
  "spotify_access_token": "your-token-here"
}

🎯 Example Questions You Can Ask

  • "What are my top 5 artists?"
  • "Show me my top 10 tracks"
  • "What's currently playing?"
  • "What songs did I listen to recently?"
  • "What playlists do I have?"
  • "Search for songs by Taylor Swift"

🔧 API Endpoints

  • GET / - This home page
  • GET /login - Spotify authentication
  • GET /callback - OAuth callback (automatic)
  • POST /ask - Ask questions with AI + Spotify
  • GET /spotify/tools - Available Spotify tools
  • GET /docs - Interactive API documentation
